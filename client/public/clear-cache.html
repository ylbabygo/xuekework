🧹 Railway错误修复工具

彻底清理浏览器缓存，解决Railway API错误

🚨 错误说明

错误：net::ERR_FAILED https://web-production-6102.up.railway.app/auth/login

原因：浏览器缓存了旧的Railway API配置，需要彻底清理。

调试信息：
正在检查浏览器状态...

📋 手动清理步骤（如果自动清理无效）

  1. Chrome/Edge: 按 Ctrl+Shift+Delete → 选择"全部时间" → 勾选所有选项 → 清理数据
  2. Firefox: 按 Ctrl+Shift+Delete → 选择"全部" → 勾选所有选项 → 立即清理
  3. 开发者工具方法: 按 F12 → Network标签 → 右键刷新按钮 → "清空缓存并硬性重新加载"
  4. 隐私模式测试: 打开隐私/无痕模式窗口测试应用
  5. 重启浏览器: 完全关闭浏览器后重新打开
  6. 检查扩展: 禁用所有浏览器扩展后测试

🔧 高级修复方法

  • 重置浏览器: Chrome设置 → 高级 → 重置并清理 → 将设置还原为原始默认设置
  • 清理DNS缓存: 命令行运行 ipconfig /flushdns (Windows) 或 sudo dscacheutil -flushcache (Mac)
  • 检查hosts文件: 确保没有railway.app相关的条目
